# Comment installer et jouer au jeu ?

## Avec InteliJ Idea

- [Installer IntelliJ](https://www.jetbrains.com/fr-fr/idea/download)
- Ouvrir le dossier depuis IntelliJ 
- Cliquer sur la flèche verte en haut a droite depuis le fichier **src/LabyrinthMainApp.java**

## Avec des lignes de command 

- Ouvrir le dossier **src**
- Ecrivez "javac LabyrinthMainApp.java" pour compiler le programme
- Puis pour lancer le programme : "java LabyrinthMainApp"
